# MVP Execution Plan (30 Days): BudgetPilot

## Week 1

- Define UX flows for onboarding, add transaction, dashboard
- Finalize data model and API contracts
- Prepare component library and design tokens

## Week 2

- Implement auth and profile
- Implement transactions CRUD
- Seed default categories

## Week 3

- Implement budget limits and warning indicators
- Build dashboard cards and charts
- Add transaction filters and search

## Week 4

- Implement reminders
- QA pass and bug fixing
- Deploy beta build and collect feedback

## Exit Criteria

- User can complete onboarding in under 2 minutes
- User can add transaction in under 10 seconds
- Dashboard reflects current month totals correctly
- No blocker bugs in core flows
