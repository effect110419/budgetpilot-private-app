# Roadmap: BudgetPilot

## Phase 0 - Foundation (Week 1)

- Finalize PRD and information architecture
- Design low-fidelity wireframes
- Set up repository, environments, CI baseline

## Phase 1 - MVP Build (Weeks 2-5)

- Authentication and profile
- Transactions CRUD
- Categories and budgets
- Dashboard and monthly reports
- Push/email reminders

## Phase 2 - Beta Release (Weeks 6-7)

- Closed beta with first users
- Fix usability issues
- Add crash and performance monitoring

## Phase 3 - Monetization (Weeks 8-9)

- Premium plan gating
- Subscription billing integration
- Upgrade flows and trials

## Phase 4 - Growth (Weeks 10-12)

- Referral mechanics
- Lightweight onboarding optimization
- Content and ASO experiments
