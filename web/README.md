# BudgetPilot Web MVP

This folder contains the first working web version of BudgetPilot.

## Features

- Add income and expense transactions
- Define expense categories
- Set monthly category budgets
- View month summary (income, expense, balance)
- See expense analytics by category
- Data saved locally in browser (`localStorage`)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Current MVP stores data locally in the browser.
- Next step is backend + authentication + cloud sync.
